**PannHarvestEngine** is a high-performance, modular resource gathering framework for Minecraft. It replaces vanilla mining and interaction mechanics with a versatile **HP-based** harvesting system suitable for ores, timber, and custom agriculture.

Featuring per-tool damage scaling, **multi-file configuration**, and a "packet-based" visual interface, it provides a seamless RPG-style experience with smooth progress bars and custom loot tables—all without interfering with vanilla block physics until the harvest is complete.

**Full ItemsAdder support included! The plugin is designed to work perfectly standalone, but automatically integrates with ItemsAdder if installed.**

----

### ✏️ Fully customizable
- Set the number of clicks for vanilla blocks and custom blocks using ItemsAdder 
- Adjust the force of your tools, vanilla and custom
- Customize technical settings such as cooldowns, sounds, the progress bar, and much more!
